## 节点启动 和 维护
> 说明 上面生成的 runc.sh 就是 如果需要分开多个账号 只需把 runc.sh 改为其他的文件名即可
- 功能 启动节点 和 Kill 节点
- 1.启动所有节点 ./[sh文件名] start
- 2.结束单个节点进程 ./[sh文件名] kill [客户端ID]
- 3.结束所有节点 ./[sh文件名] killall
